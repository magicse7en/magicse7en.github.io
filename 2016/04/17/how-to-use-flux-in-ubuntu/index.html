how to use flux in ubuntu

Posted on 2016-04-17   |   In Ubuntu   |  

f.lux是一款出色的护眼软件，可根据所在地的日出日落时间动态调节色温，过滤蓝光。一旦用上它，你会发现就再也无法离开它了。

Install fluxgui

然后运行fluxgui, 勾选上”Autostart f.lux indeicator applet”，填写好所在地的经纬度，设置完成。
此时发现flux并没有正常work. 还需要安装flux.

Install flux

上面安装的仅仅是一个GUI, 核心的flux还需要重新安装下。
64bit版本：xflux64.tgz
32bit版本：xflux32.tgz
然后解压，将解压出来的xflux拷贝到/usr/bin/下即可。
再设置fluxgui，即可生效。

#f.lux #Ubuntu
Windows10上打开ubuntu bash shell
